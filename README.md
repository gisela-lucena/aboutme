README - About Me

📖 About the Project
About Me is the first independent project of the TripleTen Web Development course. This project consists of creating a personal web page that presents information about you and your hobbies, using the HTML and CSS fundamentals learned during the first sprint.

🎯 Project Objectives

Apply basic knowledge of semantic HTML
Implement fundamental CSS styling
Create a responsive and well-structured web page
Develop skills in interpreting design scripts
Practice organizing files in web projects

🌟 Project Concept

The project is based on the structure of the "Four Design Rules" page developed during the lessons, but with customized content. Students should choose four important aspects of their lives to present, such as:

Hobbies and interests
Family and friends
Career and goals
Personal passions
### Example Structure
In the example provided, Megan chose to present:

1. Music - Her musical passion
2. Friends - Importance of friendships
3. Family - Family values
4. Web Development - New career
